**Data types** specifies size and type, could be either boolean or numeric or string. once a data type is specified you cannot assign a value of different type to that variable.


*boolean type* is declared with the keyword bool and can only receive values true or false with false as default if no value was assigned.


*numeric type* stores only integers of whole numbers. and this type is of two category: signed declared with the int keyword and can store both negative and non-negative. unsigned declared with the uint keyword and can only store non-negative values.
the type to choose depends on the value you have to store.


*float type* store positive or negative numbers with decimal points. is declared with either the keywords float32 or float64 depending on the size of the value to be stored in the variable


*string type* stores sequence of characters inside double quotes. is declared with the keyword string.
