There are *three* functions to print outputs in go:
Print()
Println()
Printf()


*the print()* prints the output with the original format, to add a whitespace or insert a new line you add as arguments " ", the new line character respectively.


*the println()* prints output with with space and a newline.


*the printf()* formats the arguments and then prints with the formatting verb such as %v, %T etc.
