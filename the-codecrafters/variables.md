A **variable** is a container for storing value.


* There are two ways of declaring a variable the first by use of the keyword var followed by name and type. second is the use of :=, in this case it is impossible to declare the variable without giving it a value. while the first method declaration can be done outside and inside a function.


* Declaring multiple variable:
multiple variables can be declared on one line without specifying the type, it is inferred. but it is impossible if type is inferred.


* When naming variable it can be short or descriptive and should be consistent with go naming scheme.
it should not contain any go keyword nor begin with a digit, can only begin with letter or the underscore character
