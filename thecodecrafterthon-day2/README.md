***This is a base converter tool, able to convert from hexodecimal or binary to decimal, and from decimal to both hex and binary.***

**How to run program**
After inputing the go run command with base converter file name in the terminal, you input the base you wish to convert from i.e dec, hex or bin, and then input the corresponding base value. Program continues running, prompting for user input until it is brought to a stop using the quit command. If stuck use the help command to see possible action.
